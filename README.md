# 💻 Full-Stack Web Portfolio & Management System

This project is a dynamic, full-stack web portfolio designed to showcase my projects, skills, and professional background. It features a custom-built Administrative Dashboard that allows for real-time content management, data persistence, and secure control over the entire platform.

🔗 **Live Demo:** [https://melihakbalik.free.nf/?i=1](https://melihakbalik.free.nf/?i=1)

---

## 🚀 Key Features

### 🌐 Client Side (Portfolio)
* **Dynamic Project Showcase:** Automatically fetches and displays the latest projects and case studies directly from the database.
* **About & Skills Section:** Features dynamic skill representations and professional summaries that can be updated instantly.
* **Contact Form:** A fully functional and secure messaging system for visitors and potential recruiters to reach out.

### 🔐 Administrative Dashboard (Admin Panel)
* **Secure Authentication:** Session-based secure login system to restrict access and protect the management panel.
* **Content Management (CRUD):** Full administrative privileges to Add, Read, Update, and Delete portfolio items, skills, and personal information on the fly.
* **Message Center:** A centralized inbox to view incoming inquiries, manage read/unread statuses, and keep track of client communications.

---

## 🛠️ Tech Stack & Architecture

* **Backend:** PHP (Handles dynamic routing, secure session management, form processing, and server-side logic)
* **Database:** MySQL (Relational database design for robust and structured data storage)
* **Frontend:** HTML5, CSS3, JavaScript / Bootstrap (Modern, clean, and fully responsive user interface)
