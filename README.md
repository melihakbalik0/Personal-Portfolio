# 💻 Full-Stack Dynamic Web Portfolio & Management System

This project is a comprehensive, full-stack dynamic web portfolio designed to showcase my software engineering projects, technical skills, and professional background. Built from scratch using modern web technologies, it bridges academic foundations with production-ready full-stack architecture.

🔗 **Live Demo:** [https://melihakbalik.free.nf/?i=1](https://melihakbalik.free.nf/?i=1)

---

## 🛠️ Technical Architecture & Tech Stack

The system integrates all layers of web development covered throughout the academic semester, completely built with native technologies without heavy framework dependencies:

* **Frontend (Client-Side):** * **Semantic HTML5 & Advanced CSS3:** Structured entirely with semantic tags, utilizing the **CSS Box Model, Flexbox, and Grid** layout configurations for a fully responsive interface tailored to all screen sizes.
    * **Vanilla JavaScript & DOM Manipulation:** Handles dynamic UI elements, active event listening, and structural DOM manipulation based on user interactions.
* **Backend (Server-Side Logic):** * **Native PHP:** Manages dynamic routing, structural server-side business logic, security protocols, and state handling.
* **Database (Data Persistence):** * **MySQL:** Relational database management system designed with structured schemas for storing project entities and incoming client messages.

---

## 🚀 Key Features & Implementation Details

### 1. Client-Side Interactivity & Form Validation (JavaScript/DOM)
* **Dynamic UI Components:** Features interactive, non-refreshing frontend components managed entirely through native JavaScript event handlers.
* **Strict Form Validation:** All contact forms are validated client-side via JavaScript/DOM manipulation to capture formatting errors and prevent empty or invalid submissions before any server request is fired.

### 2. Server-Side Logic & AJAX Integration (PHP/MySQL)
* **Asynchronous Data Loading (AJAX):** Leverages asynchronous requests (**Fetch API**) to dynamically load project details and data chunks directly from the MySQL database without inducing a full page refresh.
* **Contact Management System:** A dedicated "Contact Me" section that sanitizes and securely logs visitor inquiries directly into a structured database table.

### 3. State Management & Admin Dashboard Persistence
* **Secure Authentication:** Features a protected Administrative Panel guarded by session-based authentication to prevent unauthorized back-end access.
* **State Persistence:** Utilizes secure PHP **Sessions and Cookies** to securely maintain the user's logged-in state across pages.
* **Content Management (CRUD):** A fully operational back-end panel allowing the administrator to dynamically Add, Read, Update, and Delete project listings and skills seamlessly.
